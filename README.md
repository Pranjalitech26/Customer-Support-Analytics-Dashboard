# Customer Support Analytics Dashboard

## Project Overview
Created a Customer Support Analytics Dashboard in Excel to analyze ticket volume, urgency trends, resolution timelines, and support performance metrics.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Functions

## Key Metrics
- Total Tickets
- High-Urgency Tickets (141)
- Average Resolution Time (2.4 Hours)

## Key Insights
- Analyzed 500+ customer support tickets.
- Evaluated ticket volume, urgency trends, and resolution timelines.
- Improved reporting efficiency through interactive visuals and dashboard filters.
